# Phase 83.3 Plan: Sortformer Transformer Encoder Path

## Goal

Bind and execute the maintained `te.*` transformer encoder stack through fixed-profile native
stage code under `src/emel/diarization/sortformer/transformer/`.

## Tasks

1. Add fixed-profile transformer tensor binding for all maintained `te.l0.*` through `te.l17.*`
   tensors.
2. Validate exact maintained dimensions:
   - hidden dimension: `192`
   - feed-forward inner dimension: `768`
   - attention heads: `8`
   - max frame count: `188`
3. Add caller-owned transformer workspace for heap-free stage composition.
4. Add deterministic native kernels for layer normalization, self-attention, residual handoff,
   ReLU feed-forward, and output normalization.
5. Add focused transformer lifecycle tests for binding, rejection, deterministic kernel output,
   and invalid frame-shape handling.
6. Wire the new source and tests into CMake and run focused plus full quality gates.

## Constraints

- Keep code in `src/emel/diarization/sortformer/transformer/`.
- Do not add a generic runtime dispatcher.
- Do not route Sortformer through generation/text code.
- Do not update benchmark or snapshot baselines.
