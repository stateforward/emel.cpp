# Phase 86 Plan: ARM Sortformer Profiling Baseline

## Goal

Build an ARM profiling baseline for the maintained Sortformer path with stable end-to-end and
per-stage measurements.

## Tasks

1. Add stage-attribution timing to the maintained `diarization_sortformer` benchmark suite.
2. Report feature extraction, encoder, modules/cache, transformer, output conversion, and
   end-to-end timing in benchmark metadata.
3. Include hotspot ownership guidance in the metadata and docs.
4. Validate with the suite-filtered benchmark, focused tests, docs generation, and quality gates.

## Verification

- `git diff --check` for touched files.
- Build `bench_runner`.
- `EMEL_BENCH_SUITE=diarization_sortformer ... bench_runner --mode=emel`.

  full gate.
