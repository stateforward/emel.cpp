# Phase 185 Plan 01

## Objective

Define and enforce the ownership boundary before changing runtime behavior.

## Tasks

1. Trace current tensor, loader, and weight-loader responsibilities from source, tests, docs, and
   CMake.
2. Define `model/tensor` as the owner of tensor residency and `model/loader` as the orchestration
   layer.
3. Keep future `emel/io` strategy implementation out of scope.
4. Preserve no-queue RTC actor constraints while planning the cutover.

## Verification

  `src/emel/model/weight_loader`.
- `rg -n "weight_loader|load_weights|model/weight_loader|WeightLoader" src tools tests docs CMakeLists.txt`
