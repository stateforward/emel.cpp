# Phase 186 Plan 01

## Objective

Move bulk tensor residency behavior into `model/tensor` without regressing existing per-tensor
operations.

## Tasks

1. Add tensor-owned `bind_storage`, `plan_load`, and `apply_effect_results` events.
2. Add explicit `_done` and `_error` outcomes for bulk residency phases.
3. Keep dispatch bounded and non-allocating by using spans and action-local data-plane loops.
4. Add tests for successful bind/plan/apply, invalid input, capacity failure, effect errors, and
   guard branch coverage.
