# Phase 194 Plan 01

<rule_constraints>
- AGENTS.md no-allocation-during-dispatch rule is enforced by tasks 1-4 moving GGUF KV arena and
  entry vector growth before `model_loader.process_event(...)`.
- AGENTS.md source-backed closeout rule is enforced by task 5 with a regression test that scans the
  maintained generation benchmark, Sortformer benchmark, embedded-size probe, and paritychecker
  source paths directly.
- AGENTS.md benchmark/snapshot rule is acknowledged for task 6; the user explicitly approved
  updating snapshots, benchmarks, and models if validation requires it.
- AGENTS.md dirty-worktree quality-gate rule is enforced by task 6 using
  `EMEL_QUALITY_GATES_CHANGED_FILES` scoped to this phase's changed files.
</rule_constraints>

## Objective

Close the v1.22 audit gap by ensuring maintained tool lanes size GGUF KV storage before entering
the model-loader actor dispatch, while the parse callback only binds preallocated storage, parses,
copies tensor names, and populates metadata.
