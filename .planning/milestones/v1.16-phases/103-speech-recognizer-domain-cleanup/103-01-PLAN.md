# Plan 103.1: Speech Recognizer Domain Cleanup

## Goal

Remove the top-level `src/emel/whisper/**` runtime domain and move the existing Whisper encoder
and decoder actors under speech recognizer ownership.

## Tasks

1. Move encoder/decoder actor files to `src/emel/speech/recognizer/whisper/**`.
2. Update namespaces and includes to `emel::speech::recognizer::whisper::*`.
3. Update tests and benchmark runner includes/usages.
4. Verify no `src/emel/whisper` directory or `emel/whisper` include remains.
5. Run focused Whisper build/tests.

## Success Criteria

- No top-level `src/emel/whisper/**` tree exists.
- No include path uses `emel/whisper/...`.
- Existing encoder/decoder tests compile and preserve current behavior.
