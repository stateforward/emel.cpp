- `scripts/check_domain_boundaries.sh` exits 0.
- `rg -n "emel/whisper|namespace emel::whisper|kernel/whisper|kernel::whisper" src tests CMakeLists.txt`
  returns no matches.
- `test ! -d src/emel/whisper` passes.
- `tools/bench/whisper_emel_parity_runner.cpp` uses public Whisper `any.hpp` surfaces and
  encoder/decoder `sm.hpp`; the detail-header grep against that file returns no matches.
- `scripts/bench_whisper_compare.sh --skip-reference-build --skip-emel-build` reports
  `status=exact_match reason=ok`.
- `build/whisper_compare/summary.json` records exact `[C]` parity with model SHA
  `9ade048c9d3692b411572a9a8ad615766168e62fb1d4c234973825a377c71984` and audio SHA
  `695ac1b2c85a0419b6ee052ef90cd09cd0c5688a1445aea735b66883d199e803`.

  `speech/encoder/whisper+speech/decoder/whisper+speech/tokenizer/whisper`.
- `build/whisper_compare_tools/whisper_benchmark_tests` passes: 10 test cases and 139 assertions.
- `scripts/bench_whisper_single_thread.sh --skip-reference-build --skip-emel-build` reports
  `benchmark_status=ok reason=ok`; `build/whisper_benchmark/benchmark_summary.json` records EMEL
  mean `62836680 ns` and reference mean `66490097 ns`.

Contradictions:

- `POLICY-01`: `tools/bench/whisper_emel_parity_runner.cpp` validates and publishes
  `language/task/timestamp/suppress`, but constructs `event::decode` with only
  `std::span<const int32_t>{policy.prompt_tokens}`. `src/emel/speech/decoder/whisper/events.hpp`
  has no decode-policy field. `src/emel/speech/encoder/whisper/detail.hpp` always calls
  `select_greedy_timestamp_aware_token`, so timestamp behavior is selected by hardcoded detail
  logic instead of explicit policy input routed through SML.
- `TOK-02`: the maintained compare transcript is tokenizer-backed, but
  `src/emel/speech/encoder/whisper/detail.hpp` still exposes `write_token_transcript`, which writes
  a `token:<id>` placeholder into the decoder transcript buffer.
- Nyquist: phases 94-102 are preserved v1.16 phase directories with SUMMARY and VERIFICATION
  artifacts but no VALIDATION artifacts while Nyquist validation is enabled.
