# Phase 64 Plan 01

## Objective

Move the existing C++ reference lane onto the shared manifest-driven compare architecture without
leaking setup or runtime state into the EMEL path.

## Tasks

1. Finish the C++ reference runner contract wiring so JSONL mode emits only canonical compare
