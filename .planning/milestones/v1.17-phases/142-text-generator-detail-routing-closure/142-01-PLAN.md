# Plan 142-01: Text Generator Detail Routing Closure

<goal>
Close the `TEXTGEN-04` audit blocker by moving generator route-support predicates out of
`detail.hpp` helper outputs and into guard-owned code while preserving existing behavior.
</goal>

<tasks>

## 1. Move Route Predicates To Guards

Files:
- `src/emel/text/generator/guards.hpp`
- `src/emel/text/generator/prefill/guards.hpp`
- `src/emel/text/generator/detail.hpp`

Actions:
- Add guard-owned route predicate helpers for preselected argmax, q8 input, packed q8 input, chunk4
  packed/q8-k readiness, and chunk8 q8-k readiness.
- Update generator and prefill guards to call guard-owned predicates only.
- Keep detail helpers limited to numeric/data-plane or already-selected execution support.

## 2. Update Component-Private Tests

Files:
- `tests/text/generator/action_guard_tests.cpp`
- `tests/text/generator/detail_tests.cpp`

Actions:
- Move behavior-selection assertions from detail helper names to guard-owned predicate names.
- Keep detail tests only for data-plane helpers still owned by `detail.hpp`.

## 3. Add Source-Backed Regression

Files:
