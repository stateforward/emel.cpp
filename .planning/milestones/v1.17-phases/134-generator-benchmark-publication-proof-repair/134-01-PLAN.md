# Phase 134 Plan: Generator Benchmark Publication Proof Repair

<rule_constraints>
- The maintained EMEL publication path must drive generator behavior through public
  `emel::text::generator::sm` events.
- Actor-internal stage attribution is not valid proof unless exposed through a public, source-owned
  contract.
- Preserve output compatibility where possible while labeling unsupported attribution truthfully.
</rule_constraints>

## Tasks

1. Add a source regression that fails if `measure_emel_stage_probe` or
   `capture_generation_stage_probe` reaches into generator actor internals.
2. Simplify EMEL stage probing so it measures public `event::generate` total time and optional
   non-actor prompt-conditioning metadata only.
3. Remove or disconnect direct calls to `emel::text::generator::detail`, `action`, and guard
   helpers from the maintained publication probe path.
4. Rebuild bench tests and run generation benchmark snapshot compare.
