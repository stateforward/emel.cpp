# Phase 245 Plan

## Goal

Expose cooperative async loading through maintained public strategy contracts without allowing tools
or model loader code to claim async execution unless the public async runtime path actually ran.

## Tasks

1. Add a public `cooperative_async` strategy value to `emel::io::loader::event::strategy_kind`.
2. Route that strategy through `io/loader` as an explicit unsupported public strategy until the
   maintained async execution path and performance evidence land.
3. Teach the maintained model-load strategy helper to parse and report `cooperative_async`.
4. Add tests proving model-loader and tools see only public strategy contracts and no async actor
   internals.
5. Run focused loader tests and changed-file scoped quality gates.
