  Task 4.

# Phase 213 Plan 01: Read Validation and Platform Gating

## Goal

Add explicit read/copy validation and platform-gating states to `src/emel/io/read` so no
file-open or read attempt is structurally reachable until request, file path/file index,
length, layout, target-buffer, and platform preconditions pass.

## Requirements Closed

| REQ-ID | Source-Backed Evidence Plan |
|--------|------------------------------|
| READ-02 | `src/emel/io/read/guards.hpp` contains validation predicates and `src/emel/io/read/sm.hpp` orders them before `state_read_attempt_decision`; doctests prove representative invalid and unsupported-resource outcomes. |
| PLAT-01 | `EMEL_IO_READ_PLATFORM_SUPPORTED` and platform guards route unsupported platforms to `unsupported_platform`; supported-platform requests reach only the placeholder read-attempt state until Phase 214. |
