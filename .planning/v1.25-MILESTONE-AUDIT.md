## Source-Backed Maintained Path

Confirmed live wiring:

1. `src/emel/io/events.hpp` defines the shared tensor load span contract.
2. `src/emel/io/read` owns source-span validation and bounded source-span copy.
3. `src/emel/io/loader` exposes `load_tensor_batch` and dispatches one public
   `io/read` batch event for the already-selected `read_copy` route.
4. `src/emel/model/loader` prepares request-owned `io_load_spans`, dispatches one
   public `io/loader` batch event, and marks `used_io_strategy` only after batch success.
5. Maintained generation, Sortformer, embedded probe, and paritychecker lanes resize and
   pass request-owned `io_load_spans` before model-loader dispatch.
6. Maintained lanes use `emel::io::source::load_file_bytes` for setup-time source bytes
   and propagate public `used_io_strategy` evidence.
