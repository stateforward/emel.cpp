# Phase 59.3 Plan 01

## Objective

Raise maintained single-thread ARM image throughput again by removing the remaining pointwise
lowering seam around the current hotspot, then rebenchmark and reprofile the maintained image lane.

## Tasks

1. Add the phase artifacts and the minimum bind-time/runtime data needed to support a direct HWC
   pointwise path without dispatch-time allocation or orchestration changes.
2. Replace the maintained image pointwise tiled pack/unpack path with a direct dense HWC matrix
   path that uses one-time transposed pointwise weights and writes image outputs in-place in the
   caller-visible HWC layout.
3. Rebuild, rerun the maintained image benchmark, and resample the image-only case to record the
   throughput delta and the next ROI target after the pointwise seam is removed.
