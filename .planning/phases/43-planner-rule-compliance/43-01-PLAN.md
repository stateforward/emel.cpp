# Phase 43 Plan 01

## Objective

Bring the planner family into AGENTS-compliant state/guard/effect naming, keep transition tables in
destination-first form, and remove hidden runtime path selection from mode wrapper member
functions.

## Tasks

1. Rename planner-family top-level and mode state symbols to `state_*`.
2. Rename planner-family runtime predicates to `guard_*` and transition effects to `effect_*`.
3. Update mode wrappers so they drive their own SML models through `planner_event::request_runtime`
   and publish typed outcome events from explicit machine end states rather than hand-written branch
   trees.
4. Refresh planner-family tests to the renamed surfaces.

## Verification

- `cmake --build build/zig --target emel_tests_bin -j4`
- `./build/zig/emel_tests_bin --no-breaks --source-file='*tests/batch/planner/*'`
- `scripts/quality_gates.sh`

## Done Condition

Planner-family code reads with AGENTS-compliant `state_` / `guard_` / `effect_` naming, mode
wrappers no longer hide runtime path selection in member functions, and focused planner validation
still passes.
