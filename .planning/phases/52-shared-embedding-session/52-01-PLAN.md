# Phase 52 Plan 01

## Objective

Prove that the maintained TE text, image, and audio lanes already share one deterministic
embedding-session contract by consolidating common test scaffolding and adding trimodal contract
coverage over the existing `embeddings/generator` actor.

## Tasks

1. Extract shared canonical payload helpers in `tests/embeddings/te_fixture.hpp` so the maintained
   image and audio anchors are synthesized once and reused across modality and contract tests.
2. Add a dedicated shared-session proof file that drives text, image, and audio through the same
   embedding generator instance and verifies:
   - normalized output on the same result contract
   - uniform supported truncation at `1280/768/512/256/128`
   - uniform invalid truncation rejection on an unsupported dimension
3. Keep the phase bounded to proof and helper cleanup, then run focused embedding verification and
   full repo quality gates until everything is green.

## Verification

- `cmake --build build/zig --target emel_tests_bin -j1`
- `./build/zig/emel_tests_bin --no-breaks --test-case='*shared contract*'`
- `./build/zig/emel_tests_bin --no-breaks --source-file='*tests/embeddings/*'`
- `scripts/quality_gates.sh`

## Done Condition

EMEL proves that text, image, and audio all publish embeddings through one deterministic shared
contract with the same normalization, truncation, and invalid-dimension behavior, while the
maintained slice stays synchronous, one-modality-per-request, and repo-wide gates remain green.
