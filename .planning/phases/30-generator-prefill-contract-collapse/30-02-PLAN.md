# Phase 30 Plan 02

## Objective

Collapse the top-level prefill routing matrix around shared compute/result states while keeping
route choice explicit in the transition table.

## Tasks

1. Replace the repeated prefill state family with shared contract-decision and compute-result
   states.
2. Route concrete compute requests through explicit guard rows that stamp the chosen contract and
   invoke the existing contract-specific request action.
3. Collapse prefill success/error handling into shared result rows keyed off compute outcome and
   chosen contract.
4. Refresh generator architecture docs and topology tests for the collapsed prefill surface.

## Verification

- `cmake --build build/zig --parallel --target emel_tests docsgen`
- `ctest --test-dir build/zig --output-on-failure -R generator_lifecycle_tests`
- `scripts/generate_docs.sh`

## Done Condition

The top-level generator no longer repeats the old eight-lane prefill matrix as separate
success/error families, and route choice remains explicit in guards/transitions.
